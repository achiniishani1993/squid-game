# squid-game
Build responsive squid game - Colloboration Project

A React-based mini game inspired by Squid Game, where players register, log in, choose a character, play the game and complete on a leaderboard.

# Project Overview
This project simulates a Squid Game experience in the browser.

Users can:
- Create an account
- Log into the account
- Select a character
- Play the game
- Earn scores 
- View leaderboard rankings against random players
All user data is stored using localStorage, and random opponents are fetched from an external API.

# Features

1. Authentication

- User Registration

- User Login

- Form validation

- Password visibility toggle

- Error handling

- Audio background music on auth pages

2. Player Selection

Choose from 5 Squid Game characters


# Game Mechanics

- Doll random rotation logic

- Player movement tracking

- Score calculation

- Timer countdown

- Win / Lose state

- Auto navigation after result

# Leaderboard

- Real user score tracking

- Random generated competitors

- Rank sorting

- Highlight current user

- Dynamic score display
